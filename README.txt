# MONEY TRANSFER API

Esta API se ejecuta en un contenedor docker, proporciona funcionalidades para gestionar cuentas y transferencias de dinero entre estas
haciendo uso de una base de datos mysql dockerizada.

## Desarrollo
Esta API fue desarrollada en un entorno Windows 10, haciendo uso de las siguentes herramientas:

- IDE: IntelliJ IDEA
    - Lenguaje de desarrollo Java
    - Framework de desarrollo Java JDK 17.0.7

- Maven para inyección de dependencias
    - Apache Maven 3.9.2

- Framework spring boot 3.0.6 (https://start.spring.io/)
    - MySQL driver
    - Spring Web
    - Spring Data JPA
